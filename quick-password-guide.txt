WHERE TO FIND YOUR PASSWORD:

1. Go to MongoDB Atlas
2. Click "Database Access" (under Security)
3. Find your user (username like "admin")
4. Click pencil icon (edit) → "Edit Password"
5. Create new password and SAVE IT!

OR create a new user:
- Database Access → Add New Database User
- Create password
- SAVE IT!

Then use that password to replace <password> in connection string.
