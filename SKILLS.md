# Technologies & Skills Used

Complete list of technologies, frameworks, and skills used in this AI Chatbot project.

---

## 🎨 Frontend Technologies

### Core Framework
- **React 18** - Modern UI library for building interactive interfaces
- **Vite** - Fast build tool and dev server
- **React Router** - Client-side routing for SPA

### Styling
- **Tailwind CSS** - Utility-first CSS framework
- **PostCSS** - CSS processing
- **Autoprefixer** - Automatic vendor prefixing

### State Management
- **React Hooks** - useState, useEffect, useContext
- **Local Storage API** - Client-side data persistence
- **Custom Hooks** - useChat, useTheme, useLocalStorage

### UI/UX Features
- **Responsive Design** - Mobile-first approach
- **Dark Mode** - Theme switching
- **Animations** - Smooth transitions and typing indicators
- **Auto-scroll** - Automatic message scrolling

## ⚙️ Backend Technologies

### Core Framework
- **Node.js** - JavaScript runtime
- **Express.js** - Web application framework
- **ES Modules** - Modern JavaScript module system

### Database
- **MongoDB** - NoSQL database
- **Mongoose** - MongoDB object modeling
- **MongoDB Atlas** - Cloud database hosting

### Authentication & Security
- **JWT (JSON Web Tokens)** - Token-based authentication
- **bcryptjs** - Password hashing
- **Helmet** - Security headers
- **CORS** - Cross-origin resource sharing
- **express-rate-limit** - API rate limiting

### Validation
- **express-validator** - Input validation and sanitization

### Utilities
- **Axios** - HTTP client for API calls
- **UUID** - Unique identifier generation
- **dotenv** - Environment variable management
